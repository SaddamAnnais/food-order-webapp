
# food-order-webapp

A simple food order web application.

## Run Locally

Clone the project

```bash
  git clone https://github.com/SaddamAnnais/food-order-webapp.git
```

Go to the project directory

```bash
  cd food-order-webapp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
